# react-native-live-text-image-view
Live text api for image view
## Installation

```sh
npm install react-native-live-text-image-view
```

## Usage

```js
import { LiveTextImageViewView } from "react-native-live-text-image-view";

// ...

<LiveTextImageViewView color="tomato" />
```
